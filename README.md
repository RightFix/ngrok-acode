# ngrok-acode
Run official ngrok in acode  

# Install
```bash
apk update && apk upgrade
apk add git
git clone https://github.com/RightFix/ngrok-code

cd ngrok-acode
bash install.sh
```
# How to use it
And run : `ngrok`
#
check version: `ngrok version`
#
to configure  : `ngrok config add-authtoken <token>`
#
run server: `ngrok http 80`
